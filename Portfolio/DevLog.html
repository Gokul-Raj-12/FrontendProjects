Portfolio Build Process

being lifted. The grid lines are as subtle as they can be can be a statement to the responsive design which the user
can play with.


About Portfolio Alternate:

I wanted to emulate the artifact ability introduced in LLM's like Claude and DeepSeekCoderV2 models.
but I had problems with how to adapt to the mobile resolutions as the artifact part would take up 1/3rd of
the screen and that would make the site look slightly cluttered. So I took the idea and had to give up on
replicating it. But I translated that in my current site by combining Inline Framesand Overlay Effects.
